## ✨ Overview

**Neurochirurgie DEP** is a proposed digital patient record framework designed specifically for neurosurgical practice.

The project aims to standardize clinical documentation, streamline patient management, and create structured data suitable for clinical research and future AI-powered applications.

---

## 🎯 Goals

* 📋 Standardized patient documentation
* 🧠 Comprehensive neurological assessment
* 🩻 Structured imaging reporting
* ⚙️ Surgical workflow support
* 📊 Research-ready data collection
* 🤖 Future AI integration

---

## 🚀 Key Features

| Feature                     | Description                        |
| --------------------------- | ---------------------------------- |
| 📑 Clinical Documentation   | Organized patient records          |
| 🩺 Neurological Examination | Structured examination workflow    |
| 🩻 Imaging Module           | CT, MRI and imaging documentation  |
| ⚙️ Surgical Records         | Pre-op, intra-op and post-op notes |
| 📈 Follow-up Tracking       | Longitudinal patient monitoring    |
